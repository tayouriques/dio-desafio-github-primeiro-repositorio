## Comandos básicos aprendidos :computer:

Cd - Para navegar dentro da basta

Ls - Para listar

Ctrl + L - Para limpar a área de trabalho

Git init - Para criar um repositório

ls -a - Para mostrar arquivos ocultos

Git status - para saber o status daquele arquivo

Git add nomedoarquivo/ Git add*: vai pegar arquivos untrackeds ou unmodifieds/modifieds e vai colocar para staged

Git commit -m “…” - Faz o commit com uma mensagem e vai para o repositório
